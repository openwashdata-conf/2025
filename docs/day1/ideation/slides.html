Data Package & Analysis

Historical Data Digitization

Project: Convert scanned historical records into digital data

  • Collect historical papers/records
  • Create organized image folders
  • Use Claude to extract and analyze data
  • Create structured datasets
  • Build data packages for archival data

Open Wash Data Package Workflow

Project: Create new analyses using existing packages

  • Clone existing openwashdata package
  • Use Claude to write analytical articles
  • Add personal notes and templates
  • Create vignettes demonstrating usage
  • Share insights with community

SFD Report Analysis

Project: Analyze Shit Flow Diagram reports

  • Collect all SFD PDF reports
  • Use Claude to extract data sources
  • Categorize as primary/secondary

    • Focus group discussions
